# Дорожная карта и цели

## Текущая версия
- MVP каталога на Next.js 14 с русскоязычными гайдами по ОАЭ.
- SEO-слой: `generateMetadata`, sitemap, robots, JSON-LD.
- UX: фильтры, поисковые подсказки, TOC, похожие гайды, медиа-секции.

## Краткосрочные цели (1–2 недели)
1. **Контент**: наполнить `data/guides-ru.ts` новыми локациями (Абу-Даби, Шарджа), добавить реальные изображения.
2. **SEO**: уточнить `SITE_URL`, добавить Open Graph изображения и Twitter-карточки.
3. **Инфраструктура**: внедрить CI lint/test, устранить мастер-настройку ESLint, подготовить GitHub Actions.
4. **AI-подготовка**: стандартизировать FAQ/sections (JSON schema), добавить экспорт данных для чат-бота.

## Среднесрочные цели (1–2 месяца)
1. Локализация EN: параллельный массив `guidesEn`, переключатель языка, i18n маршруты.
2. Импорт контента из CMS (например, Sanity/Contentful) с fallback на `data/*.ts`.
3. Добавить статистику взаимодействия (анонимные события), сохранить приватность.
4. Поддержка пользовательских отзывов и рейтингов.

## Долгосрочные направления
- Полноценный AI-бот с генерацией маршрутов и ответов на основе структурированных данных.
- Мобильное приложение или PWA с офлайн-кешем гайдов.
- Монетизация: премиальные подборки, партнёрские ссылки, персонализированные рекомендации.
