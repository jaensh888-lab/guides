# Yoots Atlas Guides

Yoots Atlas — это каталог туристических гайдов для путешественников по ОАЭ и последующим направлениям. В MVP реализованы карточки гайдов, фильтры/поиск, страница отдельного гайда, базовый SEO и заглушка для будущего AI-чата.

## Стек

- Next.js 14 (App Router, TypeScript)
- Tailwind CSS
- Мок-данные в `data/guides-ru.ts`

## Локальный запуск

```bash
npm install
npm run dev
```

После старта приложение доступно на [http://localhost:3000](http://localhost:3000).

## Структура

- `app/` — страницы и маршрутизация (главная, динамические гайды, sitemap, not-found)
- `components/` — UI-компоненты (карточки, фильтры, поиск, AI-виджет, шаблон страницы гайда)
- `data/` — мок-данные для русской локали
- `types/` — типы доменной модели гайда
- `lib/` — вспомогательные функции выборки данных
